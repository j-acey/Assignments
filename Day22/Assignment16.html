SSA Java/JavaScript 12 Week Class - Team C

The Team C Java/JavaScript Class


Things I like about this class:

*opportunities to learn new concepts
*peer encouragement
*different perspectives on approach
*chill setting and atmosphere
*campus location
*instructor applying class feedback into lecture presentation

Things I think would improve the class:

*more peer collaboration
*longer breaks
*more TA lectures/participation
*after lunch, a 2-minute evaluation about the morning's lecture
*visual examples, perhaps showing a video
*allowing the students to create an assignment
